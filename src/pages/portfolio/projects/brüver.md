**Proyecto Brüver** Desarrolle una aplicación web completa para una marca de cuidado de mascotas (Brüver), aplicando tecnologías Full Stack y metodología ágil. Creando citas y reparto de horarios junto a la organización de correos.

## Tecnologías

- React
- Node.js
- JavaScript
- Zod
- Vite
- Css
- HTML
- BCrypt
- Nodemailer
- Express
- Slack
- Trello
- Figma
- Mysql
- Scrum
- Git Hub/Git

## Objetivo

Este proyecto consiste en el desarrollo de una aplicación web Full Stack para Brüver, una marca enfocada en el cuidado de mascotas. La plataforma está diseñada para gestionar citas y organizar horarios, además de facilitar la comunicación con clientes mediante correos, todo con una estructura modular y escalable. Se ha trabajado siguiendo metodología ágil (Scrum) para planificar, coordinar tareas y entregar funcionalidades de forma iterativa.
